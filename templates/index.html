Welcome to The Chennai Metro

In this website, you can explore four key features designed to enhance your experience. First, you'll find a detailed map view for every metro station included in the Phase 1 projects, allowing you to easily locate stations. Second, you can access real-time train timings for each station, ensuring you know when the next train will arrive. Third, there's an open map for navigation, offering a broader view for route planning. Lastly, you can dive into the source code to understand how these features are implemented.
